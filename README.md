# Mini Application de Réalité Augmentée avec Intelligence Artificielle / Réalisé par : Asmaa TALAL et Khouloud BENNOUBY

Ce projet consiste en le développement d’une application de réalité augmentée (AR) sous Unity en utilisant le SDK Vuforia.

L’application permet de détecter une image réelle à l’aide de la caméra, puis d’afficher un contenu virtuel en superposition. Une logique d’intelligence artificielle simulée est intégrée afin d’afficher dynamiquement une information en fonction de l’objet détecté.

## Fonctionnalités principales

- Détection d’image (Image Target) avec Vuforia  
- Affichage de contenu virtuel (texte / objet 3D)  
- Interaction en temps réel avec la caméra  
- Simulation d’une intelligence artificielle (affichage conditionnel)  

## Technologies utilisées

- Unity  
- Vuforia Engine  
- C#  

## Objectif pédagogique

Ce projet vise à comprendre :
- Le fonctionnement de la réalité augmentée  
- L’intégration d’un SDK AR (Vuforia)  
- La gestion des interactions en temps réel  
- L’intégration d’une logique d’intelligence artificielle simple
